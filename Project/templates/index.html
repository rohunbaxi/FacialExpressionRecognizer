Facial Expression Recognition Model

Check your mood with a trained neural network that analyses your facial expression. Due to Heroku's load limitations, download the open-source code from Github and deploy yourself in order to upload your own videos and check facial expressions using a better trained CNN. Make sure your videocam is turned on!

Created by Rohun Baxi using Python, Keras, and NumPy, and deployed using Flask.
The trained model recognizes faces and captures expressions with an accuracy of about 84%
